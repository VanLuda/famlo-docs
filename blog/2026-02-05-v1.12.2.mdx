## Improvements

- Added a "Use current location" button to discovery maps so you can quickly find nearby groups and events
- Map controls are now larger and easier to tap on mobile devices
- Filter button on maps is now more visible with improved styling
- Pull-to-refresh on the My Groups page now uses a smoother animation

---

## Bug Fixes

- Fixed issue where the mobile bottom sheet would overlap map interactions when panning or zooming
- Fixed fallback greeting in new device login emails
