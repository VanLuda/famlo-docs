This release features a redesigned homepage with a parent-focused marketplace layout, event timezone support across all listing pages, and the foundation for Spanish language support.

<!-- truncate -->

---

## What's New

### Redesigned Homepage
The homepage has been completely reimagined with a parent-focused marketplace layout, making it easier to discover groups, events, and resources at a glance.

### Event Timezone Support
Events now display timezone information on all listing pages, so you always know the correct local time for activities regardless of where they are hosted.

### Spanish Language Support
We have begun rolling out Spanish language support across the platform, starting with authenticated areas including the feed, post composer, account settings, and dashboard.

### Events and Classes Renamed
The main navigation now leads with discovery, and "Events" has been renamed to "Events & Classes" to better reflect the variety of activities available.

---

## Improvements

- State event pages now display event counts with thousand separators for easier reading
- Group and event images now load with blur placeholders for a smoother visual experience
- Improved error messages when connectivity issues occur
- Added URL-based pagination to state and city event pages

---

## Bug Fixes

- Fixed horizontal overflow on the homepage
- Fixed issue where unauthenticated visitors could experience unnecessary loading delays
