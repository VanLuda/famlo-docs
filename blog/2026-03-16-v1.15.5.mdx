---

## What's New

### Event Subscriptions
Group leaders can now set up recurring billing for events, making it easy to manage weekly classes, monthly meetups, and other repeating activities with automatic payment collection.

### Redesigned Homepage Hero
The homepage hero section has been refreshed with a new design to better showcase the platform's value for homeschool families.

---

## Improvements

- Event cards are now fully clickable for easier navigation
- Improved loading states with auth-aware skeleton animations
- Group creation flow has been polished for a smoother setup experience
- Improved header layout and navigation
